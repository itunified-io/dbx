## dbxcli db

Oracle database read-only operations

### Synopsis

Read-only Oracle database operations covering sessions, tablespaces,
users, schemas, SQL execution, redo logs, undo, parameters, and advisors.

Requires a target with an Oracle database endpoint configured.

### Options

```
  -h, --help            help for db
      --target string   target name (from ~/.dbx/targets/)
```
