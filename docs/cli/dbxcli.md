## dbxcli

dbx — multi-database management platform

### Synopsis

dbx is a multi-database management platform supporting Oracle and PostgreSQL.

Commands use named parameters (emcli-style):
  dbxcli <domain> <action> entity_name=<name> entity_type=<type> [key=value ...]

### Options

```
      --format string   output format: table, json, yaml (default "table")
  -h, --help            help for dbxcli
```
